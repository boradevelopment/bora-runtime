# BORA Runtime
Before using, install 3rdparty libs via 3rdparty/prebuild.bat.
## Please locate builds at root/builds directory!
BORA Runtime is based on WAMR. Although BORA will be planned to integrate with WAMR and have it become apart of the repository instead of as a third party. That would take significant time.

For building, just go to CLion and build. Compile through the SDK and test.
